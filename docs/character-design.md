# 萌娃跑步角色设计

## 设计原则

角色不直接使用照片原图，而是从照片和文字设定中提炼可识别特征，统一成适合移动端跑酷的萌版卡通形象。第一版仍使用 Phaser 几何图形绘制，后续可以替换为正式 2D 角色素材。

## 角色清单

### 小弟弟

- 人设：半岁男宝宝。
- 关键词：纸尿裤、圆脸、飞行帽、护目镜。
- 视觉提炼：参考图里有浅绿色帽子和护目镜，因此保留“飞行宝宝”的趣味点，同时保持纸尿裤宝宝设定。
- 游戏表现：大头小身体，浅绿色上身，白色纸尿裤，头顶护目镜。

### 小星星

- 人设：幼儿园中班女孩。
- 关键词：西瓜头、背带裤、星星。
- 视觉提炼：参考图里孩子穿浅色上衣，整体很日常；游戏里加强成黑色圆润西瓜头和蓝色背带裤，并加星星胸标。
- 游戏表现：短圆发、蓝色背带裤、黄色星星装饰。

### 冰雪小公主

- 人设：小学一年级女孩，萌版冰雪公主风格。
- 关键词：浅金发、长辫、蓝白裙、皇冠、雪花。
- 视觉提炼：保留冰雪、公主、皇冠、蓝白配色，但不直接复刻任何受版权保护角色。
- 游戏表现：蓝白裙、浅金头发、雪花皇冠、白色星雪装饰。

### 小布丁

- 人设：小学二年级女孩。
- 关键词：粉色裙、甜美、彩虹帽。
- 视觉提炼：参考图有粉色双辫和彩虹帽，和“粉色裙”设定很适配；第一版保留帽子、粉裙和甜美元素。
- 游戏表现：粉色裙、双侧圆发、彩虹帽、小星星点缀。

### 空军上尉

- 人设：小学五年级男孩。
- 关键词：空军制服、飞行员、战机徽章。
- 视觉提炼：参考图是战机，因此游戏角色不画真实战机，而是转化为空军制服、飞行帽、翼形徽章。
- 游戏表现：蓝灰制服、军帽、肩章、胸前星章和小飞机翼形标识。
